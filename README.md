📘 Java Exercises – Repo di Mauro Bilardo
Questa repository è una raccolta delle lezioni e esercitazioni di java fatte durante il corso con Masamune
sono presenti i concetti base della programmazione a oggetti e dei pattern per la creazione di un progetto java


🎯 Obiettivi del Repository
Consolidare i concetti fondamentali della programmazione in Java

-Applicare correttamente i principi OOP
-Utilizzare i design pattern in modo pulito, modulare e riproducibile
-Prepararsi a progetti più complessi e a contesti professionali
-Mantenere uno stile di codice chiaro, didattico e facilmente estendibile

📂 Struttura della Repository
La repository è organizzata in cartelle tematiche, ognuna dedicata a un argomento specifico:

Basi:  
Variabili, tipi di dato, operatori, input/output, controllo di flusso.

OOP:  
Classi, oggetti, ereditarietà, incapsulamento, polimorfismo, astrazione.

Patterns:
Implementazioni didattiche dei principali design pattern:(Singleton,Decorator,Observer,MVC,Facade,Strategy)
